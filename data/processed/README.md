# Processed Dataset Output

This directory stores normalized files generated by:

```powershell
python tools/prepare_tcm_finetune_data.py --input data/raw/chatmed_tcm/train.jsonl --output data/processed/tcm_finetune.jsonl
```

Generated records use this JSONL shape:

```json
{
  "messages": [
    {"role": "system", "content": "..."},
    {"role": "user", "content": "..."},
    {"role": "assistant", "content": "{\"preliminary_assessment\":\"...\"}"}
  ],
  "metadata": {
    "source": "chatmed_tcm",
    "policy": "no_medicine_no_prescription",
    "commercial_risk": true
  }
}
```

For commercial use, prefer processed data produced from self-owned or explicitly commercial-licensed source cases.
